1. 삼쩜삼의 4가지 약점 (1차 출처 검증)

약점 1: 공정위 과징금 ₩7,100만원 (거짓 광고)

"새 환급액 도착" 알림에서 실제로는 0원이 다수. 평균 환급금 53만원 광고와 달리 실제는 17만 5천원 (2023 김영선 의원실 국감 자료). YTN 2025-12-28 보도.

약점 2: 개인정보위 과징금 ₩8억 5,410만원

2023년 6월 개인정보위 발표. 주민등록번호 무단 수집·보관. 비즈한국 보도.

약점 3: 국세청 '원클릭' 무료 출시 (2025-03-31)

311만 명에게 ₩2,900억 환급 안내 (경향신문). 핵심 비즈니스 모델 직접 위협.

약점 4: 세무사회 분쟁 진행 中

1차 사건은 2025-05-29 대검 재항고 기각으로 무혐의 (한국경제). 단 2024년 11월 '삼쩜삼TA' (세무사 명의 신고) 별건은 수사 중.

2. 국세청 원클릭 — 진짜 무료 환급 방법

원클릭 절차 (5분)

  1. 홈택스 (hometax.go.kr) 접속
  2. 로그인 (공인인증서·간편인증)
  3. "종합소득세 신고" → "모두채움 신고" 또는 "원클릭"
  4. 자동 채워진 정보 확인 (3.3% 원천징수·인적공제·세액공제)
  5. 제출 → 환급 계좌 입력 → 신고 완료

원클릭 환급금 받는 일정

  • 5월 31일까지 신고 → 6월 말 ~ 7월 환급
  • 지방세 환급 = 별도 (시군구청 처리)
⚠️ 5월 31일 미신고 시: 무신고 가산세 20%·납부 지연 가산세 추가. 환급 대상이라도 5년 이내 경정청구 가능하나, 5월 정시 신고가 가장 효율적.

5. 자주 묻는 질문

Q1. 직장인도 부업 환급 받을 수 있나요?

예. 단 인적공제 중복 시 추징 위험. 본업 회사가 인적공제를 이미 처리했으므로 부업 신고 시 인적공제 다시 X.

Q2. 5년 전 못 받은 환급은?

경정청구 5년 가능. 2021년분까지 청구 가능. 국세청 원클릭에서 자동 안내됨.

Q3. 토스 세이브잇·카카오 세이브투게더는?

10~17% 수수료. 토스는 추징 안심보상 100만원 제공·카드 등록 의무. 무료 = 국세청 원클릭만.

Q4. 환급액이 0원이면?

3.3% 떼인 총액 < 산출세액 = 환급 X. 추가 납부 의무 발생 가능. 우리 계산기에서 사전 확인 가능.

Q5. 사업자등록 없이 신고 가능?

인적용역 사업소득자 (3.3% 원천징수) = 사업자등록 의무 X. 사업소득자 등록 (홈택스·5분)만으로 신고 가능.
