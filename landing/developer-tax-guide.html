7. 자주 묻는 질문

Q1. 직장인 부업 개발 외주는?

본업 인적공제·부업 종소세 합산 의무. 부업 환급액 = 약 ₩50~150만.

Q2. 1인 SaaS 메이커 (앱 매출 직접)는?

사업자 등록 (간이과세 ₩8천만 미만) + 종소세 + 부가세. 사업자 등록 후 모든 클라우드·도구·외주 비용 처리·매출 큰 시 사업자 의무.

Q3. 해외 클라이언트 (Upwork·Stripe)는?

외화 매출 = 영세율 (0%)·종소세 신고 의무. 원화 환산 후 신고. PayPal·Wise 영수증 보관.

Q4. AWS·GCP 비용 어떻게 처리?

매월 정기 결제·연 평균 ₩수백만 비용. 카드 명세 또는 클라우드 인보이스 100% 처리. 사업자 등록 시 부가세 환급 가능.

Q5. GitHub Copilot·Cursor 등 AI 도구 비용?

업무 관련 = 100% 비용 처리. 개인용 ChatGPT Plus도 업무 사용 입증 시 50~100% 처리 가능.
