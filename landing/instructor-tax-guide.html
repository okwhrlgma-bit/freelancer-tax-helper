2. 강의료 부가세 면제 (부가가치세법 §26)

부가가치세법 제26조에 따라 "교육 용역"은 부가세 면세입니다. 강사료·강의료 = 부가세 0%·세금계산서 X.
  • 학원·교습소 강의료: 부가세 면세
  • 외부 출강 강사료: 부가세 면세 (인적용역)
  • 온라인 강의 (인강·라이브): 부가세 면세
  • 기업 출강·B2B 교육: 부가세 면세
  • 교재 인세: 부가세 면세 (저작권법)
⚠️ 예외 (과세): 학습용 도구 판매·강의 영상 USB 판매 등 = 부가세 과세 (재화 매출)

7. 강사 종소세 신고 절차

  1. 홈택스 → 종소세 신고 → 모두채움 또는 정기 신고
  2. 업종코드 (940903 또는 921505) 입력
  3. 총 수입 입력 (학원·플랫폼 별·원천징수 차감 전 금액)
  4. 경비 입력 (장부 또는 단순경비율 75.5%)
  5. 인적공제·세액공제 확인
  6. 제출 → 환급 계좌 입력 → 완료

8. 자주 묻는 질문

Q1. 학원에서 강의 + 외부 출강 같이 하는 경우?

두 수입 모두 종소세 합산. 학원 = 940903, 외부 = 921505 모두 포함. 경비율은 합산 수입에 따라 결정.

Q2. 인프런 강의 수익 세금?

3.3% 원천징수 + 종합소득세 신고 의무. 5월 신고 시 환급 가능 (총 세금 < 원천징수 시). 인프런 지급명세서 = 홈택스 자동 반영.

Q3. 교재 인세 수입은?

저작권 인세 (940904) = 별도 신고. 출판사 원천징수 (3.3%) 후 입금·종소세 신고 시 환급 대상.

Q4. 학원 사업자 vs 1인 강사?

학원 = 사업자 등록 의무·다수 강사 고용 시. 1인 강사 = 인적용역으로 가능·연 ₩7,500만 미만 사업자 등록 X 가능.

Q5. 직장인 부업 강의는?

부업 수입 = 종합 수입에 합산. 본업 회사 인적공제 우선·부업 신고 시 중복 X.
