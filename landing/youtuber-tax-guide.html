3. 유튜버 비용 처리 가이드

가능한 경비 항목

  • 장비비: 카메라·마이크·조명·삼각대·컴퓨터·편집 소프트웨어
  • 편집 외주비: 영상 편집·썸네일·자막·번역 (세금계산서 의무)
  • 회의·미팅비: 협찬사·광고주 미팅 카페·식대 (영수증)
  • 구독료: 어도비·캔바·노션 등 도구
  • 교통비: 촬영 출장·콘텐츠 제작 이동 (영수증)
  • 홈오피스 임차료: 전체 임차료의 30~50% (촬영 공간 비율)
  • 인터넷·전기료: 50% 안분

4. 해외 광고 수익 부가세 면제 (애드센스)

구글 애드센스 등 해외 광고 수익 = 외화 매출·영세율(0%) 적용·but 종합소득세 신고 의무 그대로.
  • 달러 수입 → 원화 환산 → 종합소득세 신고
  • 부가세 = 영세율·환급 0원·but 신고 의무
  • 구글 세금 정보 (W-8BEN-E) 등록 의무·미국 원천징수 회피

5. 사업자 등록 시점

사업자 등록 의무 트리거

  1. 수익이 계속·반복 발생 (애드센스 입금 시작)
  2. 면세 (940306) 또는 과세 (921505) 결정
  3. 홈택스 → 사업자등록 → 5분 내 발급
✅ 1인 미디어 (940306) 면세 권장: 부가세 신고 의무 X·종소세만
✅ 편집자 고용 시 (921505) 과세: 부가세 신고 의무·매출 ₩8천만 미만 간이과세

6. 유튜버 종소세 신고 절차 (60분)

  1. 홈택스 (hometax.go.kr) 로그인 (간편인증)
  2. "종합소득세 신고" → "모두채움 신고" 또는 "정기 신고"
  3. 업종코드 입력 (940306 또는 921505)
  4. 총 수입 입력 (애드센스·협찬·후원 등 합산·원화 환산)
  5. 경비 입력 (장부 또는 단순경비율)
  6. 인적공제·세액공제 확인
  7. 제출 → 환급 계좌 입력 → 완료

7. 자주 묻는 질문

Q1. 구독자 1만 명 이하·수입 적은데 신고 해야 하나요?

예. 수입 1원이라도 신고 의무. 단순경비율 적용 시 환급 가능 多.

Q2. 구글 애드센스 외 후원 (수퍼챗·멤버십)은?

모두 종합 수입에 포함. 후원금 = 사업소득. 멤버십 = 정기 구독 사업소득.

Q3. 협찬·광고비는?

광고주에서 세금계산서 발행 시 = 일반 사업소득. 무계산서 = 직접 신고.

Q4. 영상 편집 외주비는 100% 경비 처리되나요?

예. 단 세금계산서 또는 사업자등록증 사본 + 계좌이체 영수증 의무.

Q5. 직장인 부업으로 유튜브하는 경우?

본업 회사가 인적공제 처리·부업 신고 시 인적공제 다시 X (중복 추징 위험).
