# 서비스 수준 약정 (SLA·Service Level Agreement)

> 30-apps 공유 SLA 템플릿·각 앱이 사업자 정보·서비스명 채워서 사용.
> B2B 영업·기관 (도서관·재단·자치구) 요구 시 즉시 제출 가능.

## 1. 서비스 수준 (Service Levels)

### 1.1 가용성 (Uptime)

| 플랜 | 월 가용성 목표 | 월 허용 다운타임 |
|---|---|---|
| Free | best-effort (99.0%) | 약 7시간 |
| Pro | **99.5%** | 약 3시간 30분 |
| Founding | **99.5%** | 약 3시간 30분 |

→ Streamlit Cloud 기반 = 인프라 가용성 의존·계획 정비 시간 = 사전 7일 공지 후 가용성 계산 제외.

### 1.2 응답 시간 (Performance)

| 영역 | 목표 |
|---|---|
| 페이지 로드 (p95) | ≤ 3초 |
| API 응답 (p95) | ≤ 1초 |
| 결제 처리 (p99) | ≤ 10초 (PG 의존) |

### 1.3 데이터 보존 (Retention)

- 결제 기록 = 5년 (전자세금계산서 의무·국세청)
- audit log = 6개월 (PIPA 5만명+ 처리 의무)
- 사용자 핵심 데이터 = 사용자 컴퓨터 한정 (헌법 §14·서버 저장 X)
- 백업 = 결제·audit만 (사용자 데이터 X)

## 2. 지원 (Support)

### 2.1 채널

- 이메일 (1순위·우리 = 1인 운영·실제)
- 처리방침 페이지 문의 양식

### 2.2 응답 시간

| 플랜 | 응답 SLA |
|---|---|
| Free | 5 영업일 내 best-effort |
| Pro | **2 영업일 내** |
| Founding | **1 영업일 내** + 우선순위 |

### 2.3 영업 시간

- 평일 09:00~18:00 KST (한국 표준시)
- 주말·공휴일 = 응답 X (24시간 모니터링 = 별도 약정 시)
- 야간 긴급 = 결제·로그인 차단만 (배포 자동 알림)

## 3. 보안 약정 (Security)

| 영역 | 정합 |
|---|---|
| 전송 암호화 | TLS 1.2+ (Streamlit Cloud·PG 자동) |
| 비밀번호 저장 | bcrypt rounds=12 (OWASP 권장) |
| API 키 | .env only (헌법 §3·하드코딩 X) |
| audit log | 해시 체인 SHA-256 (PIPA 5/5) |
| 로그인 | brute force 방어 (5회/5분·15분 락) |
| 결제 | webhook HMAC + idempotency + order tampering 검증 |
| PII 노출 | 화면·로그 마스킹 (PIPA 정합) |
| 침투 시험 | 연 1회 (외부 발사 후·기간별 변동) |

## 4. 사고 대응 (Incident)

### 4.1 분류

| 등급 | 정의 | 1차 대응 |
|---|---|---|
| **P0 결제 차단** | 모든 결제 실패 | 1시간 내 알림 |
| **P1 로그인 차단** | 가입·로그인 X | 4시간 내 알림 |
| **P2 부분 장애** | 일부 기능 미작동 | 영업일 1일 내 |
| **P3 표시 오류** | 화면 일부 깨짐 | 영업일 5일 내 |

### 4.2 PIPA 72h 신고 의무

- 1,000명+ 정보 유출·해킹·민감정보 = 정보주체 + KISA 72시간 내 신고
- 사고 발생 = 즉시 audit chain 백업·증거 보존
- 정직 = "수정 후 통지" X·**유출 시 = 즉시 통지** (대법원 판례)

## 5. 환불 (Refund)

전자상거래법 §17 정합 (별도 `refund_policy_kr.md` 참조):

- 7일 내 + 미사용 (≤30%) = 전액 환불
- 7일 내 + 사용 (>30%) = 부분 환불 (사용 차감)
- 7일 초과 = 거부 (단, 시스템 장애 = 별도)

## 6. 데이터 이관 (Portability)

- PIPA §35의2 = 사용자 요구 시 = 표준 형식 (CSV·JSON) 30일 내 제공
- 형식: 사용자 본인 데이터·결제 기록 (audit 제외·민감정보 마스킹)
- 비용: 무료 (연 1회·이후 인쇄·매체 비용 청구 가능)

## 7. 약정 미달 시 (Service Credit)

| 미달 영역 | 보상 |
|---|---|
| Pro 가용성 < 99.5% | 다음 달 10% 할인 |
| Pro 응답 시간 미달 (반복) | 다음 달 5% 할인 |
| P0 사고 4시간+ | 한 달 무료 |

→ 자동 산정 X·사용자 신청 시 검토 후 적용 (CS 메일).

## 8. 계약 변경 (Amendments)

- 본 SLA 변경 시 = 사전 30일 공지 (마이페이지·이메일)
- 이의 시 = 환불 + 해지 가능
- B2B 별도 계약 = 본 SLA를 base로 협의 (영업 문의)
