# 환불 정책 template (전자상거래법 §17·30-apps 공유)

> 사용처: 모든 30-apps 배포 시 자동 inject (Cycle 111)
> 정합: 전자상거래법 §17·소비자보호법·한국 No.1 환불국 패턴
> 핵심: 7일 무조건 환불·1클릭 취소·월할 환불

---

## 1. 환불 가능 기간

### 1-1. 7일 무조건 환불 (전자상거래법 §17)
- 결제 후 **7일 이내** = 사용 여부 무관·**무조건 환불**
- 사유 X·자유 환불·100% 반환

### 1-2. 7일 이후 환불
- 월정액 = **월할 환불** (미사용 일수 비율)
  - 예: 30일 중 10일 사용 → 20/30 = 66.7% 환불
- 일회성 결제 = 7일 이후 환불 X (사용 시작 후 디지털 콘텐츠)

## 2. 환불 신청 방법

### 2-1. 1클릭 자동
- 사용자 = 마이페이지 → "구독 취소" 클릭
- 즉시 자동 갱신 정지·환불 처리
- 7영업일 이내 = 결제 수단으로 반환

### 2-2. 이메일 신청
- `support@<your-domain>` 또는 운영자 이메일
- 24시간 이내 응답 (best-effort)

## 3. 환불 불가 사유

| 사유 | 근거 |
|---|---|
| 7일 경과 + 일회성 결제 | 전자상거래법 §17 ②항 |
| 사용자 약관 위반 (다중 계정·악용) | 전자상거래법 §17 ②항 |
| 외부 PG (PortOne·Stripe) 환불 거부 | 위탁자 정책 |

## 4. 자동 갱신 알림

### 4-1. 14일 전 알림 (의무)
- 자동 갱신 14일 전 = 이메일 알림
- 회원 = 14일 내 = 1클릭 취소 가능

### 4-2. 결제 후 영수증
- 결제 즉시 = 영수증 이메일 발송 (Resend)
- 세금계산서 = 사업자 회원 = 별도 요청·홈택스 자동

## 5. PG별 환불 처리 시간

| PG | 환불 처리 |
|---|---|
| PortOne (한국) | 7영업일 이내 (체크카드·계좌이체) |
| Stripe (글로벌) | 5~10영업일 (국제 카드) |
| Lemon Squeezy (MoR) | 7영업일 (글로벌·VAT 위임) |

## 6. 분쟁 해결

1. **1차**: 운영자 협의 (이메일·고객 지원)
2. **2차**: 한국 소비자원 (1372)
3. **3차**: 전자거래분쟁조정위원회·서울중앙지방법원

## 7. 정합 정책

- 전자상거래법 §17 (청약 철회·7일)
- 소비자보호법
- 한국 No.1 환불국 정합 (DelightRoom 분석)
- ADR 0058 정합 (캐시카우 통과 앱 = 본 정책 의무)
