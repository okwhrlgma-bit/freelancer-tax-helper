# 쿠키 정책 (Cookie Policy)

> 본 문서 = 30-apps 공유 템플릿·각 앱이 사업자 정보·문의처 채워서 사용.
> 정합: PIPA·전자상거래법·정보통신망법 §50의5 (쿠키 사전 동의 의무 부분).

## 1. 본 정책의 목적

본 정책은 본 서비스가 사용자 단말에 저장·접근하는 **쿠키(Cookie)**·**유사 기술 (localStorage·sessionStorage)** 의 종류·목적·보관 기간·관리 방법을 안내합니다.

## 2. 쿠키란

쿠키 = 웹사이트 접속 시 사용자 브라우저에 저장되는 작은 텍스트 파일. 사용자 식별·로그인 유지·환경 설정·이용 통계 등에 사용됩니다.

## 3. 사용하는 쿠키 (3 종류)

### 3.1 필수 쿠키 (사전 동의 X·서비스 운영 의무)

| 이름 | 목적 | 보관 |
|---|---|---|
| `session_id` | 로그인 세션 유지 | 세션 종료 시 |
| `csrf_token` | CSRF 공격 방지 | 세션 종료 시 |

→ 본 쿠키 = 서비스 운영 필수·차단 시 = 로그인·결제 불가.

### 3.2 기능 쿠키 (선택·동의 시)

| 이름 | 목적 | 보관 |
|---|---|---|
| `lang` | 언어 설정 (ko·en) | 1년 |
| `theme` | 다크/라이트 모드 | 1년 |

### 3.3 분석·마케팅 쿠키 (옵트인 의무·정보통신망법 §50의5)

| 이름 | 출처 | 목적 | 보관 |
|---|---|---|---|
| (예: `_ga`) | Google Analytics | 사용 통계 (익명) | 2년 |

→ 본 쿠키 = 가입 시 **마케팅 동의** 별도 체크 시만 활성. 미동의 시 = 0건.

## 4. 사용자 권리 (PIPA·정보통신망법)

- 브라우저 설정 → 쿠키 차단·삭제 가능
- 마이페이지 → 동의 철회 가능 (즉시 처리)
- 본 정책 변경 시 = 사전 7일 공지·재동의 요청

## 5. 헌법 §14 정합 (사용자 데이터)

본 서비스 = 사서·사용자 핵심 데이터 = **사용자 컴퓨터 (offline·session_state)** 처리·서버 저장 X.
쿠키 = 로그인·세션·환경 설정 한정·민감정보 직접 저장 X.
