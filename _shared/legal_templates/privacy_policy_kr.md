# 개인정보처리방침 template (한국 PIPA 정합·30-apps 공유)

> 사용처: 모든 30-apps 배포 시 자동 inject
> 출처: 개인정보보호위원회 「개인정보처리방침 만들기」 + GitHub `kimlawtech/korean-privacy-terms`
> 헌법 §14 정합: 사용자 데이터 = 사용자 컴퓨터·SaaS 서버 X (offline 우선)

---

## 1. 개인정보의 처리 목적

본 서비스는 다음의 목적을 위하여 개인정보를 처리합니다:
- 회원 가입 의사 확인·서비스 제공·요금 결제·정산
- 고객 상담·민원 응대·서비스 개선

## 2. 처리하는 개인정보 항목

| 분류 | 항목 |
|---|---|
| 필수 | 이메일 |
| 선택 | 이름·연락처 |
| 자동 수집 | IP·접속 로그·쿠키·서비스 이용 기록 |

**자관·민감정보 = 본 서비스 서버 저장 X** (헌법 §14·offline 우선·사용자 컴퓨터 처리).

## 3. 개인정보의 처리 및 보유 기간

- 회원 가입 정보: 회원 탈퇴 시까지
- 결제 기록: 5년 (전자상거래법)
- 접속 로그: 3개월 (통신비밀보호법)

## 4. 개인정보의 제3자 제공

원칙적으로 제공 X. 다음 예외:
- 정보주체 사전 동의
- 법령 의무 (수사기관·세무관서)
- 결제 처리 (PortOne·Stripe·Lemon Squeezy 등 PG)

## 5. 개인정보 처리의 위탁

| 수탁자 | 업무 |
|---|---|
| PortOne (또는 Stripe·Lemon Squeezy) | 결제 처리 |
| Resend (또는 SendGrid) | 이메일 발송 |
| Plausible (옵션) | 익명 통계 (cookie-less) |

## 6. 정보주체의 권리·의무 및 행사 방법

정보주체는 언제든지 다음 권리 행사 가능:
- 열람·정정·삭제·처리정지 요구
- **자동화 결정 거부권** (PIPA 2026.3 개정안 정합·헌법 §11)
- **개인정보 전송 요구권**

행사 = 이메일 (privacy@<your-domain>) 또는 서비스 내 설정.

## 7. 개인정보의 안전성 확보 조치

- bcrypt 해시 (비밀번호)
- AES-256 암호화 (저장 시)
- TLS 1.2+ (전송 시)
- 접근 권한 분리 + audit_log 해시 체인

## 8. 자동화된 결정에 관한 사항

본 서비스는 LLM·룰 기반 추천을 제공할 수 있으며, 정보주체는 다음 권리 행사 가능:
- 자동화 결정 거부 권리
- 사람의 검토 요구
- 결정 근거 설명 요구

→ 본 서비스 권고 = 자기 측정·검토 보조이며 의료·법률·세무 자문 X.
