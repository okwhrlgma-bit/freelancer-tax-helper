# 이용약관 template (한국 전자상거래법 정합·30-apps 공유)

> 사용처: 모든 30-apps 배포 시 자동 inject (Cycle 111·ADR 0066 정합)
> 정합: 전자상거래 등에서의 소비자보호에 관한 법률 (전자상거래법)·표준약관·공정위 정합
> 헌법 §14 정합·사용자 데이터 = 사용자 컴퓨터 (offline 우선)

---

## 제1조 (목적)

본 약관은 사용자(이하 "회원")가 본 서비스(이하 "서비스")를 이용함에 있어 회사(개인사업자·1인 PO)와 회원의 권리·의무 및 책임 사항을 규정함을 목적으로 합니다.

## 제2조 (정의)

1. **서비스**: 본 운영자가 제공하는 SaaS 도구 (예: 종소세 환급 추정·N잡 추적·KORMARC 자동 등)
2. **회원**: 본 약관에 동의하고 회원 가입을 한 사용자
3. **유료 서비스**: 월정액 또는 일회성 결제 후 제공되는 서비스
4. **자기 측정 보조**: 본 서비스 = 자동 결정 X·사용자 검수 의무 (헌법 §11 정합)

## 제3조 (약관의 효력 및 변경)

1. 본 약관은 회원 가입 시 동의로 효력 발생
2. 약관 변경 = 7일 전 사전 공지 (전자상거래법)
3. 변경 후 사용 = 동의 간주

## 제4조 (회원 가입)

1. 가입 = 이메일 + 비밀번호 + 처리방침 동의 + 이용약관 동의
2. 만 14세 미만 = 가입 X (PIPA 정합)
3. 본인 = 1 계정 (다중 계정 X)

## 제5조 (서비스 제공)

1. 서비스 = 24/7 best-effort (장애 시 = 공지)
2. 데이터 = 사용자 컴퓨터·session_state (헌법 §14·offline 우선)
3. 결제 기록·계정 정보 = PG 위탁·5년 보관 (전자상거래법 §6)

## 제6조 (요금 결제)

1. 요금 = 가격 페이지 명시 (월정액 또는 일회성)
2. 결제 = PortOne·Stripe·Lemon Squeezy 중 자동 선택 (국가별)
3. 자동 갱신 = 14일 전 알림 (사전 동의)
4. 세금계산서 = 사업자 회원 한정·홈택스 자동 발급 (PortOne)

## 제7조 (환불 및 청약 철회·전자상거래법 §17 정합)

1. **7일 무조건 환불**: 결제 후 7일 이내·사용 무관 (전자상거래법 §17)
2. **월할 환불**: 월정액 = 미사용 일수 비율 환불 (한국 No.1 환불국 정합)
3. 1클릭 취소 = 자동 갱신 즉시 정지
4. 환불 = 결제 수단으로 7영업일 이내 (전자상거래법)

## 제8조 (회원의 의무)

1. 정확한 정보 입력
2. 비밀번호 보관 책임
3. 서비스를 의료·법률·세무 자문으로 사용 X (헌법 §11 정합·면책 의무)
4. 자관 데이터·민감정보 입력 X (헌법 §14 정합)

## 제9조 (운영자의 의무)

1. 안정적 서비스 제공 best-effort
2. 개인정보 보호 (PIPA 정합·처리방침 별도)
3. 환불 정책 준수
4. 자동 결정 X·사용자 검수 의무 명시

## 제10조 (책임 제한)

1. 본 서비스 = **자기 측정 보조이며 의료·법률·세무 자문 X** (헌법 §11)
2. 자동 결정 = 사용자 책임 (검수 의무)
3. 제3자 서비스 (PG·이메일·OAuth) = 위탁자 책임
4. 천재지변·전쟁·화재 등 = 면책

## 제11조 (분쟁 해결)

1. 1차 = 협의 (이메일·고객 지원)
2. 2차 = 한국 소비자원·전자거래분쟁조정위원회
3. 3차 = 서울중앙지방법원 관할

## 제12조 (지적재산권)

1. 코드 = MIT/Apache-2.0 (오픈 소스)
2. 사용자 입력 데이터 = 사용자 소유
3. 운영자 = 사용자 데이터 무단 사용 X

## 제13조 (개정 이력)

- 2026-MM-DD: 최초 제정·v1.0
